# Changelog

## 0.1.0

### Features
- Added self-hosted 3D-print asset organization with projects, tags, archive workflows and browser previews.
- Added OIDC authentication, role-based access control and file-mounted secret configuration.
- Added Docker Compose deployment with a non-root, read-only runtime container.

### Improvements
- Added public deployment, security and contribution documentation.
- Generalized deployment examples so they contain no environment-specific addresses, paths or identities.
